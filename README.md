## About
This is a command-line AI chatbot built using Python and the OpenAI GPT API. It simulates a real-world AI assistant capable of context-aware conversations, greeting detection, and graceful exit handling. The chatbot handles user inputs with robust error checking and validates responses from the API. Designed with modular code, it allows easy extension and maintenance. This project demonstrates hands-on experience in AI, Python programming, and API integration.

**Technologies:** Python, OpenAI API, CLI, GitHub

## Features
- Context-aware conversation handling  
- Greeting detection and graceful exit  
- Robust error handling and input validation  
- Modular design for easy extension  
- Interactive command-line interface
